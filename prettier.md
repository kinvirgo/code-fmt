# Prettier 配置

```sh
# 指定字符数后强制换行，默认是 80 列
printWidth : <{正整数}>

# 每个缩进级别的空格数,默认2
tabWidth : <{正整数}>

# 用制表符而不是空格缩进行,默认false
useTabs : <bool>

# 语句末尾打印分号,默认:true
semi : <bool>

# 使用单引号，默认是 false
singleQuote : <bool>

# 对象中的属性使用引号,默认： 'as-needed'
quoteProps : <as-needed|consistent|preserve>

# jsx中单引号替换双引号,默认:false
jsxSingleQuote : <bool>

# 数组对象末尾逗号
trailingComma : <es5 | none | all>

# 对象文字中的括号之间打印空格 ,默认:true
bracketSpacing : <bool>

# 将>多行 HTML（HTML、JSX、Vue、Angular）元素的 放在最后一行的末尾，而不是单独放在下一行（不适用于自关闭元素），,默认:false
bracketSameLine : <bool>

# 箭头函数参数使用圆括号包裹 比如 (x) => x 和 x => x 的区别
arrowParens : <always|avoid>

# 是否缩进Vue 文件中的代码<script>和<style>标签,默认:false
vueIndentScriptAndStyle : <bool>
```
